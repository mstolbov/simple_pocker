# Simple Poker

Texas-holdem poker game emulator

# Usage
```
Usage: bin/simple_poker [options]
Example: bin/simple_poker --players=2
    -p, --players=N                  Players count
    -n, --number=N                   Number of the games
        --log=[path]                 Path to log
    -h, --help                       Show this message
```
